Google Fit / Health Connect

Importe automatiquement ton poids, tes nuits de sommeil et tes séances depuis Google Fit (synchronisé avec Health Connect sur Android).

⚙️ Configuration requise (une seule fois)
  1. Va sur console.cloud.google.com → crée un projet
  2. Active l'API Fitness API
  3. Crée des identifiants OAuth 2.0 → Application Web
  4. Ajoute http://localhost (ou ton URL) dans les origines JavaScript autorisées
  5. Copie ton Client ID ci-dessous

⚠️ L'authentification ne fonctionne pas en file:// — utilise Live Server (extension VSCode) ou npx serve .

Importer un fichier CSV / JSON

Compatible avec les exports de Samsung Health, Garmin Connect, Google Takeout et le format interne JSON.

📂 Glisser-déposer ou cliquer pour choisir un fichier
📋 Formats CSV acceptés

Pour le CSV générique, utilise ces en-têtes (1ère ligne) :

  • Poids : date,poids,tourTaille — date au format YYYY-MM-DD
  • Sommeil : date,tempsTotal,rem,profond,leger,bpm,oxygen,apnee — durées en minutes
  • Séances : date,type,duree,kcal,ressenti,exercices
  • Repas : date,type,aliments,calories,proteines,glucides,lipides
